Detailed Description

CButton class provides a debounce button service.

Note
The CButton is a static class.
Examples:
/ble/ble_app_hid_keyboard/src/main.cpp, /ble/ble_app_proximity/src/main.cpp, and /cookbook/ble_app_hid_camera_shutter/src/main.cpp.

virtualinherited

Set the pin as an output pin.

CPin myPin(19);
myPin.output(); // set the P0.19 as an output pin. (use the default NOT_OPEN and LOW level output)
myPin = HIGH; // set P0.19 to HIGH (use the operator '=')
myPin.write(LOW); // set P0.19 to LOW (use the write() member)
Parameters
modeis a PIN_OUT_MODE_T enumeration to indicate the output mode.
defValueis a PIN_LEVEL_T enumeration to set the default level for the output pin.
