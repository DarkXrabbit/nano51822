Detailed Description

Use the CSerial class to transceiver the serial stream.

Examples:
/ble/ble_app_hrm_htm_bat/src/main.cpp, /ble/ble_app_hrm_htm_bat_lilypad/src/main.cpp, /ble/ble_app_uart/src/main.cpp, /ble/ble_beacon/src/main.cpp, /ble/ble_rssi_measure/src/main.cpp, /peripherals/adc_test/src/main.cpp, /peripherals/hw_pwm/src/main.cpp, /peripherals/i2c_scanner/src/main.cpp, /peripherals/spi_loop_test/src/main.cpp, /peripherals/sw_pwm_firefly/src/main.cpp, /peripherals/uart_echo/src/main.cpp, /peripherals/watchdog/src/main.cpp, and /rtos/blink/src/main.cpp.
